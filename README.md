A simple Kino for displaying rich links on Livebook.

## Installation

Install KinoLink in Livebook:

```elixir
Mix.install([
  {:kino_link, "~> 0.1"}
])
```

## Usage

Display a rich link on Livebook.

```elixir
KinoLink.design("Qb8fTiCItLCttOzhCE4VlF")
```
